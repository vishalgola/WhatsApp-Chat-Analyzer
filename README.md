# 📊 WhatsApp Chat Analyzer

A **data analysis web application** built using **Python and Streamlit** that analyzes exported WhatsApp chats and provides insights such as message statistics, activity timelines, emoji usage, and user activity.

This project helps visualize chat patterns and understand communication behavior in a WhatsApp conversation.

---

# 🚀 Features

* 📈 **Message Statistics**

  * Total messages
  * Total words
  * Media messages
  * Links shared

* 👥 **Most Active Users**

  * Identify who sends the most messages in group chats

* 📅 **Activity Timeline**

  * Monthly activity timeline
  * Daily activity timeline

* 🗓 **Weekly Activity Map**

  * Most active day of the week

* ⏰ **Hourly Activity Map**

  * Most active hour of the day

* ☁️ **WordCloud**

  * Visual representation of frequently used words

* 🫠 **Emoji Analysis**

  * Most frequently used emojis

# 📊 How It Works

1. Export a WhatsApp chat from your phone.
2. Upload the `.txt` chat file into the application.
3. The application processes the chat and generates various analytics and visualizations.

---


---

# 📤 How to Export WhatsApp Chat

1. Open WhatsApp.
2. Open any chat.
3. Tap **Three dots → More → Export Chat**.
4. Select **Without Media**.
5. Upload the exported `.txt` file in the app.
